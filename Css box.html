The CSS box model is a set of rules that help dictate how items are displayed within your website or web project.
you can alter every aspect of the box model even use shadows to add effect to the background

The CSS box model is a layout model that defines the rectangular boxes that are generated for elements in the document tree
and laid out according to the visual formatting model.
It describes the content of the element, the padding around the content, the border around the padding,
and the margin around the border.
